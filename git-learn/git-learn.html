1.基本操作：

添加更改文件 --> 本地提交 --> 推送远端

git add .

git commit -m 'commit info'

git pull origin 'branchname'

git push origin 'branchname'

2.新建分支：

新建本地分支 --> 切回主分支 --> 删除新建的分支

git checkout -b new

git checkout master

git branch -d new

3.新建分支并推送到远端：

git checkout -b new

git push origin new

4.合并本地分支：

合并new分支到master分支：

git checkout master

git merge new

5.删除分支

删除本地分支：git branch -d new

删除本地远程分支：git branch -r -d orgin/new

远程删除git服务器上的分支：git push origin -d new

6.版本回退

git reset --hard '版本号'

7.切换分支

git checkout dev

8.查看分支

git branch

8.gitignore修改不生效:存在缓存

git rm -r --cached .

git add .

git commit -am 'update gitignore'

git push
